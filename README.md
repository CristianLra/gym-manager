# Gym Manager

Sistema web para la gestión de rutinas de entrenamiento. Permite visualizar ejercicios por grupo muscular y guardar la última rutina seleccionada en el navegador.

## Descripción

Este proyecto es una aplicación frontend desarrollada con HTML, CSS y JavaScript. Simula una interfaz de usuario para gestionar rutinas de gimnasio de manera interactiva.

## Características

- Visualización de rutinas de entrenamiento
- Selección de grupos musculares
- Render dinámico de ejercicios
- Persistencia de datos usando localStorage
- Interfaz responsive básica
- Animaciones en la visualización de contenido

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (DOM manipulation)
- localStorage
